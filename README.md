# Генератор задач для Jira

Локальное веб-приложение на Next.js для генерации задач в Jira с помощью LLM. Приложение работает полностью на русском языке.

## Быстрый старт

1. Установи Node.js (https://nodejs.org), если еще не установлен
2. Открой терминал в папке проекта
3. Выполни команды:
   ```bash
   npm install
   ```
4. Создай файл .env.local и добавь свой API ключ:
   ```
   NEXT_PUBLIC_OPENROUTER_API_KEY=твой_ключ_от_openrouter
   ```
5. Запусти приложение:
   ```bash
   npm run dev
   ```
6. Открой в браузере Chrome: http://localhost:3000

## Как получить API ключ OpenRouter

1. Зайди на https://openrouter.ai
2. Зарегистрируйся
3. Перейди в раздел Keys
4. Создай новый ключ
5. Скопируй его в .env.local

## Использование

1. Нажми кнопку микрофона и опиши задачу голосом
2. Или введи описание текстом
3. Подтверди определенную команду
4. Ответь на уточняющие вопросы (если есть)
5. Получи готовую задачу
6. Нажми "Копировать" и вставь в Jira

## Функциональность

### Главный экран
- Голосовой ввод через Web Speech API (работает в Chrome)
- Текстовый ввод
- Автоматическое определение команды из текста
- Выбор команды и подтипа вручную

### Управление шаблонами
- Просмотр всех шаблонов команд
- Редактирование шаблонов
- Добавление новых команд
- Удаление команд
- Сброс шаблона к значению по умолчанию

### История задач
- Просмотр последних 30 созданных задач
- Копирование задачи
- Удаление из истории

## Технологии

- Next.js 14 с App Router
- TypeScript
- Tailwind CSS
- localStorage для хранения данных
- Web Speech API для распознавания речи
- OpenRouter API для работы с LLM (модель: anthropic/claude-sonnet-4)
